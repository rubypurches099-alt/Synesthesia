# Synesthesia - MIDI Visualizer Collection

**The Sound of Colour**

A collection of interactive MIDI visualizers that map musical notes to colors using Newton's color wheel and Munsell's color model.

ðŸ”— **Live Demo:** [https://rubypurches099-alt.github.io/Synesthesia/](https://rubypurches099-alt.github.io/Synesthesia/)

## Features

- Real-time MIDI to color visualization
- Multiple visualizer styles (Original, Oscilloscope, Grid, Recorder)
- Newton's color wheel mapping (C=Violet, D=Red, E=Orange, F=Yellow, G=Green, A=Blue, B=Indigo)
- Octave-based brightness control
- Extensive customization options
- No installation required - runs in browser

## Quick Start

1. Visit the [live site](https://rubypurches099-alt.github.io/Synesthesia/)
2. Choose a visualizer from the collection
3. Connect your MIDI keyboard or use the on-screen piano
4. Play and watch your music transform into color

## Visualizers

### Original Lightweight
The foundation version with Newton's color wheel mapping, chord detection, and extensive background customization options.

### Oscilloscope
Real-time waveform visualization. MIDI notes converted to sine waves at their actual frequencies with additive color mixing.

### MIDI Grid Visualizer
Grid-based visualization displaying MIDI notes in a structured layout with precise timing.

### MIDI Visualizer Recorder
Advanced visualizer with recording capabilities to capture and replay performances.

## Color Mapping

Based on Newton's color wheel from "Opticks":
- **C** - Violet
- **D** - Red
- **E** - Orange
- **F** - Yellow
- **G** - Green
- **A** - Blue
- **B** - Indigo

Octaves control brightness (C0 = darker, C10 = brighter)
Velocity affects size and saturation
