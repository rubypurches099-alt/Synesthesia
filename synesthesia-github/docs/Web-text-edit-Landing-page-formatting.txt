1. EDITING THE MAIN TITLE (WAVEFORM TEXT)
==========================================

Location: Near the bottom of the HTML file (around line 1620)

Find this code:
    // Initialize
    drawWaveformText('titleWave1', 'Welcome', false);
    drawWaveformText('titleWave2', 'To', true);  // true = small size
    drawWaveformText('titleWave3', 'Synesthesia', false);

Change to your text:
    drawWaveformText('titleWave1', 'Your Text', false);
    drawWaveformText('titleWave2', 'to', true);
    drawWaveformText('titleWave3', 'New Name', false);

IMPORTANT NOTES:
- The middle line with 'true' makes it small (like "to")
- 'false' makes it full size
- Keep text short (8-12 characters works best)
- Supported letters: W E L C O M E T O S Y N H A I (and spaces)
- Text is case-insensitive but will display as drawn

EXAMPLES:
    drawWaveformText('titleWave1', 'Hello', false);
    drawWaveformText('titleWave2', 'To', true);
    drawWaveformText('titleWave3', 'The Show', false);
    
    Result: "Hello" (big) "To" (small) "The Show" (big)

6. QUICK SEARCH TIPS
=====================

To find and edit text efficiently:

STEP 1: Open the HTML file
    - Use a text editor: VS Code, Sublime Text, Notepad++, or even Notepad
    - Right-click the file ‚Üí Open With ‚Üí Choose your editor

STEP 2: Use Find/Search
    - Windows: Press Ctrl+F
    - Mac: Press Cmd+F
    - A search box will appear

STEP 3: Search for specific text
    - Type the exact text you want to change
    - Hit Enter to jump to that location
    - Press F3 (or Cmd+G on Mac) to find next occurrence

STEP 4: Make your changes
    - Edit the text between the HTML tags
    - Don't delete or modify the tags themselves (< > / =)

STEP 5: Save and test
    - Save the file (Ctrl+S or Cmd+S)
    - Refresh your browser (F5 or Cmd+R)
    - Check your changes

SEARCH EXAMPLES:
    Want to change "Welcome"? Search for: drawWaveformText('titleWave1'
    Want to change subtitle? Search for: class="subtitle"
    Want to change "Demo" folder? Search for: <h3>Demo</h3>
    Want to change footer? Search for: class="footer"

8. PRO TIPS & BEST PRACTICES
=============================

BEFORE YOU START:
‚úÖ Make a backup copy of your HTML file
‚úÖ Name it something like "synesthesia-landing-backup.html"
‚úÖ Store it in a safe location

WHILE EDITING:
‚úÖ Change text between tags, not the tags themselves
‚úÖ Keep opening and closing tags matched
‚úÖ Don't delete < > / = " ' symbols unless you know what they do
‚úÖ Save frequently (Ctrl+S / Cmd+S)
‚úÖ Test in browser after each major change

HTML TAG BASICS:
‚úÖ <p>Text here</p> = Paragraph
‚úÖ <h3>Text here</h3> = Small heading
‚úÖ <h4>Text here</h4> = Smaller heading
‚úÖ <strong>Text</strong> = Bold text
‚úÖ <li>Text here</li> = List item
‚úÖ <a href="...">Text</a> = Link

COMMON MISTAKES TO AVOID:
‚ùå Deleting closing tags (</p>, </div>, </h3>, etc.)
‚ùå Forgetting to close quotes: onclick="showSection('demo')
‚ùå Changing text inside onclick="" or class="" attributes
‚ùå Using unsupported letters in waveform text
‚ùå Making folder names too long

TESTING YOUR CHANGES:
1. Save the HTML file
2. Open (or refresh) in your browser
3. Check all sections still work
4. Click each folder to verify it opens correctly
5. If something breaks, undo your last change

USING FIND & REPLACE:
- Most editors have Find & Replace (Ctrl+H / Cmd+Option+F)
- Useful for changing repeated text throughout the file
- Be careful - preview changes before replacing all!

RECOMMENDED TEXT EDITORS:
FREE OPTIONS:
- VS Code (https://code.visualstudio.com/) - Best for beginners
- Notepad++ (Windows only) - Simple and fast
- Sublime Text - Fast and lightweight
- Atom - User-friendly

PAID OPTIONS:
- Sublime Text (free trial, $99 license)
- PhpStorm / WebStorm - Professional tools


GETTING HELP:
=============

If something breaks:
1. Check the browser console for errors (F12 ‚Üí Console tab)
2. Restore from your backup
3. Try making smaller changes one at a time
4. Verify all quotes and brackets are matched

Common error messages:
- "Unexpected token" = You have a missing or extra bracket/quote
- "undefined is not a function" = You changed something in a script section
- Nothing displays = Check if you accidentally deleted a closing tag
